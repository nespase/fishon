Как появился fishtocli

Сидел, смотрел на погоду за окном и думал: "А клюёт ли сегодня?" В телефоне — куча приложений с прогнозом клёва, но все какие-то... не те. Хотелось простоты: открыл терминал, выбрал место — и готово.

Так и родилась идея bash-скрипта, который берёт реальную погоду с open-meteo, анализирует давление (и его перепады!) и выдаёт вердикт по каждой рыбе. Без API-ключей, без регистраций, без смс.

Первая версия появилась через пару часов. Потом ещё добавил кэш давления, потом приманки, потом тактику... А дальше — понеслось.

Гайд: установка на Windows

Скрипт работает на Linux/macOS, но на Windows его тоже можно запустить через WSL.

Шаг 1. Установка WSL

wsl --install

Эта команда установит WSL2 и Ubuntu по умолчанию. После установки перезагрузи компьютер.

Шаг 2. Запуск Ubuntu

Найди в меню "Ubuntu" и запусти. При первом запуске создай пользователя и пароль.

Шаг 3. Установка зависимостей

sudo apt update
sudo apt install curl python3 bc git

Шаг 4. Скачивание скрипта

git clone https://github.com/nespaset/fish2cli
cd fish2cli
chmod +x fish2cli.sh

Шаг 5. Запуск

./fish2cli.sh

Готово! Теперь у тебя в Windows работает настоящий рыбацкий bash-скрипт.
